**Featured Project**: Customer Shopping Behavior Analysis

The Challenge: Analyze 3,900 purchase records to identify key revenue drivers, customer loyalty segments, and the impact of subscription models on spending behavior.

**1. Data Engineering & Cleaning (Python)**
Missing Data: Handled 37 missing values in the review_rating column by imputing the median rating specific to each product category.

Feature Engineering: Created age_group bins and purchase_frequency_days to better categorize customer habits.

ETL Pipeline: Cleaned and standardized data in Python (Snake Case) before loading it into a MYSQL database for advanced querying.

**2. Strategic SQL Insights**
Customer Segmentation: Classified the user base into Loyal (3,116 customers), Returning (701), and New (83) segments.

Revenue Analysis: Identified that while there are more male customers, the Young Adult age group is the highest revenue contributor at $62,143.

Subscription Impact: Discovered that non-subscribers actually have a slightly higher average spend ($59.87) than subscribers ($59.49), suggesting a need to revamp subscription value propositions.

Discount Dependency: Flagged products like Hats (50%) and Sneakers (49.66%) as being highly dependent on discounts for sales.
